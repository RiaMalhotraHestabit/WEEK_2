Introduction

What is Semantic HTML?

Semantic HTML uses meaningful tags that describe the purpose of the content. Instead of generic containers, elements like <article>, <section>, and <nav> improve readability, accessibility, and SEO.

Common Semantic Elements

  • <header> : introductory content
  • <nav> : navigation links
  • <main> : primary page content
  • <article> : independent content
  • <section> : grouped topics
  • <aside> : supplementary information
  • <footer> : metadata and links

Accessibility

Accessibility Best Practices

Accessibility ensures content is usable by everyone, including screen reader and keyboard users.

  • Use proper labels for form inputs
  • Provide alt text for images
  • Use ARIA labels where needed
  • Ensure logical tab navigation

Quick Tips

Avoid using <div> when a semantic element fits the purpose. Always choose meaning over convenience.

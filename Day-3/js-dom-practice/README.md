## Overview
This task focuses on using **modern JavaScript (ES6)** concepts and **DOM manipulation** without any frameworks.  
The main exercise is building an **interactive FAQ accordion** using JavaScript.

---

## Learning Objectives
- Understand `let` and `const`
- Use arrow functions
- Work with arrays and objects
- Practice `map()`, `filter()`, and `reduce()`
- Manipulate the DOM dynamically
- Handle user interactions using event listeners

---

## Exercise: FAQ Accordion
An interactive FAQ accordion where:
- Clicking a question expands the answer
- Only one FAQ stays open at a time
- FAQ items are generated dynamically using JavaScript

---

## Concepts Used
- **Array of Objects** for FAQ data
- **map()** to render FAQ items
- **filter()** as a mini-challenge
- **reduce()** as a mini-challenge
- **Event Listeners** for click interactions
- **DOM Manipulation** using `querySelector` and `classList`

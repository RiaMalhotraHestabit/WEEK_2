# DAY 2 – CSS Layout Mastery (Flexbox & Grid)

## Overview
This task focuses on building a **modern, responsive UI layout** using **CSS Flexbox and CSS Grid**. The goal was to understand layout structuring, responsiveness, and real-world UI replication using core CSS techniques.

---

## Learning Outcomes
- Built responsive layouts using **Flexbox** and **CSS Grid**
- Practiced **CSS selectors and specificity**
- Designed layouts that adapt from **desktop to mobile**
- Improved understanding of layout planning and spacing

---

## Topics Covered

### 1. CSS Selectors & Specificity
- Used class selectors effectively
- Maintained clean and readable CSS structure
- Avoided unnecessary specificity conflicts

### 2. Flexbox
- Created a responsive **navigation bar**
- Built a **hero section** with aligned text and images
- Used Flexbox for alignment and spacing

### 3. CSS Grid
- Designed a structured page layout using `grid-template-areas`
- Managed multi-column layouts
- Practiced responsive rearrangement of grid items

### 4. Responsive Design
- Converted desktop layout to **mobile-friendly layout**
- Used media queries for screen adaptability
- Ensured content readability on smaller devices

---

## Outcome
- Successfully recreated the layout structure using Flexbox & Grid
- Implemented hover effects and responsive behavior
- Achieved a clean, modern UI aligned with the reference design

---

## Files Included
- `index.html` – Semantic HTML structure
- `style.css` – Layout, styling, Flexbox & Grid implementation

---

## Key Takeaway
Flexbox and CSS Grid together provide powerful control over layout design, enabling scalable, responsive, and maintainable UI development without relying on external frameworks.
